## Pulse Coaching

**PulseCoaching** est une application de gestion d'équipe de football. Elle permet de gérer les entraînements, les matchs, les joueurs, les entraîneurs, les blessures, les pénalités, les présences et les statistiques. L'application aide également à créer des équipes en fonction des joueurs disponibles, de leurs positions et de leur condition physique.
